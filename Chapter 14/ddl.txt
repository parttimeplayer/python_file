----------------------------------------------------------------------------------
单行注释： -- 注释内容（--后面一定要有一个空格）
单行注释：# 注释内容
多行注释：/*

	*/
-----------------------------------------------------------------------------------
DDL语句(Data Definition Language)：
创建数据库： Create Database 数据库名称 [charset utf8];
查看数据库： Show Databases;
使用数据库： Use 数据库名称;
查看当前使用的数据库: Select Database();
删除数据库: drop databse 数据库名称;

查看有哪些表: Show tables;
创建表:  Create Table 表名称(
	列名称 列类型，
	列名称 列类型，
	......
            );
删除表: Drop Table 表名称;
----------------------------------------------------------------------------------------------------------
DML语句(Data Manipulation Language):
插入数据：Insert into 表名称 [列1，列2，......列N] Value (值1，值2，....值N)[,(值1，值2，....值N)];
删除数据：Delete From 表名称 [Where 条件判断];
更新数据：Updata 表名称 Set 列=值 [Where 条件判断]; 
----------------------------------------------------------------------------------------------------------
DQL语句(Data Query Language)：
基础查询：Select * From 表名称 [Where 条件判断];
分组聚合：Select 字段|聚合函数 From 表 [Where 条件判断] Group by 列
【注】：Group By中出现了哪个列，哪个列才能出现在Select中的非聚合中
聚合函数：
1.SUM(列)
2.AVG(列)
3.MIN(列)
4.MAX(列)
5.COUNT(列 | *) 求数量
结果排序 分页限制：
Select 列|聚合函数|* From 表
Where...
Group By...
Order By...[ASC | DESC]
Limit n[, m];(n表示显示n条, n,m表示跳过n条取m条)
----------------------------------------------------------------------------------------------------------
